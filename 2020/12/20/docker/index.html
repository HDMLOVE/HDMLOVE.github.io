docker

Docker 本质上就是一个虚拟机。

在docker中有几个基本的概念。

1、镜像：类似于平时使用的iso或者image镜像文件。

2、容器：容器其实就会类似于虚拟机

3、tar文件：类似于vmdk文件，可以直接打个包在别的地方直接run运行。

4、docker file。docker 的一些配置

5、仓库：存放有很多已经打包好的镜像，MySQL镜像、ngnix镜像等。
